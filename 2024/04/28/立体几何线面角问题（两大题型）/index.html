【方法技巧与总结】

(1)定义: 平面上的一条斜线与它在平面的射影所成的锐角即为斜线与平面的线面角.

(2)范围: $\left[0, \frac{\pi}{2}\right]$

(3)求法:

常规法: 过平面外一点 $B$ 做 $B B^{\prime} \perp$ 平面 $\alpha$, 交平面 $\alpha$ 于点 $B^{\prime}$; 连接 $A B^{\prime}$, 则 $\angle B A B^{\prime}$ 即为直线 $A B$ 与平


可以采用等体积法求 $h$, 斜线长即为线段 $A B$ 的长度);

【典型例题】

题型一: 定义法

【典例 1-1】(2024 山东$\cdot$二模) 已知三棱雉 $P-A B C$ 中, $P A \perp$ 平面 $A B C, A B \perp A C$, 过点 $M$ 分别作平行于平面 $P A B$ 的直线交 $A C, P C$ 于点 $E, F$.

(1)求证: $E F / /$ 平面 $P A B$;

(2)若 $M$ 为 $B C$ 的中点, $P A=A B=3, A C=4$, 求直线 $P M$ 与平面 $A B C$ 所成角的正切值.
 
 
 
 
 
 
 
 
【典例 1-2】(2024-高一山西大同$\cdot$阶段练习）如图, 在四棱雉 $P-A B C D$ 中, 底面 $A B C D$ 为矩形, $P D \perp$ 底面 $A B C D, A D=P D, E, F$ 分别为 $C D, P B$ 的中点.

(1)求证: $E F \perp$ 平面 $P A B$;

(2)设 $A B=\sqrt{2} B C$, 求 $A C$ 与平面 $A E F$ 所成角的正弦值.
 
 
 
 
 
 
 
 
【变式 1-1】 (2024$\cdot$高一$\cdot$江苏$\cdot$阶段练习) 如图在四棱雉 $P-A B C D$ 中, 底面 $A B C D$ 是正方形, 侧面 $P A D \perp$ 底面 $A B C D$, 且 $P A=P D=\frac{1}{2} B D$, 设 $E, F$ 分别为 $P C, B D$ 的中点.

(1)求证：平面 $P A B \perp$ 平面 $P D C$;

(2)求直线 $E F$ 与平面 $A B C D$ 所成角的大小.
 
 
 
 
 
 
 
 
【变式 1-2】 (2024-高一$\cdot$陕西咸阳$\cdot$阶段练习) 如图, 在四棱雉 $P-A B C D$ 中, 底面 $A B C D$ 为平行四边形, $O$是 $A C$ 与 $B D$ 的交点, $\angle A D C=45^{\circ}, A D=A C=2, P O \perp$ 平面 $A B C D, P O=2, M$ 是 $P D$ 的中点.

(1)证明: $P B / /$ 平面 $A C M$;

(2)求直线 $A M$ 与平面 $A B C D$ 所成角的正切值.
 
 
 
 
 
 
 
 
【变式 1-3】 (2024-高三$\cdot$河北衡水$\cdot$期中) 在如图所示的直三棱柱 $A B C-A_{1} B_{1} C_{1}$ 中, $D 、 E$ 分别是 $B C, A_{1} B_{1}$ 的中点.

(1)求证: $D E / /$ 平面 $A C C_{1} A_{1}$;

(2)若 $\triangle A B C$ 为等边三角形, 且 $A B=A A_{1}, M$ 为 $A B$ 上的一点, $A M=\frac{1}{4} A B$, 求直线 $D E$ 与直线 $A_{1} M$ 所成角的正切值.
 
 
 
 
 
 
 
 

题型二：等体积法

【典例 2-1】 (2024-高三$\cdot$全国$\cdot$阶段练习) 如图, 在三棱台 $A B C-A_{1} B_{1} C_{1}$ 中, $A A_{1} \perp$ 平面 $A B C, \angle A B C=90^{\circ}$, $A A_{1}=A_{1} B_{1}=B_{1} C_{1}=1, \quad A B=2$.

(1)求证: 平面 $A B B_{1} A_{1} \perp$ 平面 $B C C_{1} B_{1}$;

(2)求 $A C$ 与平面 $B C C_{1} B_{1}$ 所成角正弦值.
 
 
 
 
 
 
 
 
【典例 2-2】(2024-高三$\cdot$山东菏泽$\cdot$开学考试) 如图, 在三棱柱 $A B C-A_{1} B_{1} C_{1}$ 中, $A_{1}$ 在底面 $A B C$ 上的射影为线段 $B C$ 的中点, $M$ 为线段 $B_{1} C_{1}$ 的中点, 且 $A A_{1}=2 A B=2 A C=4, \angle B A C=90^{\circ}$.

(1)求三棱雉 $M-A_{1} B C$ 的体积;

(2)求 $M C$ 与平面 $M A_{1} B$ 所成角的正弦值.
 
 
 
 
 
 
 
 
【变式 2-1】(2024$\cdot$高二$\cdot$浙江绍兴$\cdot$期末）如图, 四边形 $A B C D$ 为正方形, $E D \perp$ 平面 $A B C D, F B / / E D$, $A B=E D=2 F B=2$.

(1)求证: $A C \perp$ 平面 $B D E F$;

(2)求 $B C$ 与平面 $A E F$ 所成角的正弦值.
 
 
 
 
 
 
 
 

【过关测试】

1、 如图, 在四棱雉 $P-A B C D$ 中, 底面 $A B C D$ 是边长为 2 的正方形, $P A=a, A C \cap B D=O, P A \perp$ 底面 $A B C D$.

(1)证明: 平面 $P B D \perp$ 平面 $P A C$;

(3) 若 $\overrightarrow{P E}=\frac{1}{3} \overrightarrow{P D}$, 在线段 $B C$ 上是否存在点 $F$, 使得 $E F / /$ 平面 $P A B$, 若存在点 $F$, 则 $a$ 为何值时, 直线 $E F$与底面 $A B C D$ 所成角为 $45^{\circ}$ 。

4.、如图, 在直三棱柱 $A B C-A_{1} B_{1} C_{1}$ 中, $D$ 为棱 $A B$ 的中点, $E$ 为侧棱 $C C_{1}$ 的动点,且 $C E=\lambda C C_{1}(0<\lambda<1)$.

(1)是否存在实数 $\lambda$, 使得 $D E / /$ 平面 $A B_{1} C_{1}$ ？若存在, 求出 $\lambda$ 的值; 若不存在, 请说明理由;

(2)设 $A B=A A_{1}=4, A C=3, B C=5$, 求 $D E$ 与平面 $A B B_{1} A_{1}$ 所成角的正弦值的取值范围.

(1)请作出 $A_{1} B_{1}$ 与平面 $C D E$ 的交点 $M$, 并写出 $A_{1} M$ 与 $M B_{1}$ 的比值（在图中保留作图痕迹, 不必写出画法和理由);

(2)求直线 $B M$ 与平面 $A B C$ 所成角的正弦值.
